# Vintage Games Hub

A launcher for the Vintage Games collection. Shows all arcade and puzzle games with progress tracking.

## Features

- Grid view of all 9 games
- Achievement progress per game
- High score tracking
- Overall completion stats
- Responsive design

## Games

### Arcade
- **Hopper** - Cross the road without getting hit
- **Intercept** - Defend cities from missile attacks
- **Rescue** - Save survivors with your helicopter
- **Detonator** - Demolish buildings strategically
- **Digger** - Dig tunnels and collect gems

### Puzzle
- **Crates** - Push crates onto targets (Sokoban)
- **Lemming** - Guide creatures to safety
- **Conduit** - Connect pipes before time runs out
- **Excavation** - Dig for gems, avoid boulders
